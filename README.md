# Pacman - Unity [Demo](http://pacman-unity.herokuapp.com/)
[![Build Status](https://travis-ci.org/bosnian/pacman-unity.svg?branch=master)](https://travis-ci.org/bosnian/pacman-unity)

## About

This is personal class project for course **Computer graphics** on [FIT - Faculty of Information Technology, Mostar ](http://www.fit.ba)

It is Pacman like game, where target is to collect all coins while being chased by ghosts. Player has only three lives to collect all points.
